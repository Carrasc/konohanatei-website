# Konohanatei Website

Next.js site for **Konohanatei (此の花亭)** — Japanese café & restaurant in Jardines del Pedregal, CDMX.

Ported from the Claude Design hi-fi prototype.

## Stack
- Next.js 15 (App Router)
- React 19
- TypeScript
- Hand-authored CSS (tokens in `src/app/tokens.css`, site styles in `src/app/globals.css`)

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000
